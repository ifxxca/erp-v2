# Applications

Applications:

- `api/`: Laravel API modular monolith.
- `erp-web/`: management desktop web.
- `ops-web/`: mobile-first Operations PWA.
- `mobile/`: Flutter mobile application, dibuat setelah Operations API stabil.

Laravel API, ERP Web, dan Operations Web sudah di-scaffold tanpa menyalin source legacy. Flutter app tetap ditunda sampai Operations API stabil.
